# Images Directory

Place your high-resolution image assets here.

### Recommended Structure:
- `website/public/images/hero/` - For the main banner images
- `website/public/images/portfolio/` - For the gallery/portfolio images
- `website/public/images/about/` - For personal or studio shots

### How to use them in the code:
In `public/index.html`, look for the `<img>` tags and replace the `src` attribute.
For example, change:
`src="https://images.unsplash.com/..."`
to:
`src="images/portfolio/wedding-1.jpg"`
